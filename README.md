Proyecto: Banco - gestión de cuentas RUT y AHORRO

Instrucciones rápidas:

- Instalar dependencias:

```powershell
npm install
```

- Ejecutar servidor:

```powershell
npm start
```

- Abrir frontend en: http://localhost:3000/

Descripción:
- API REST bajo `/api` con endpoints para listar clientes, listar clientes con RUT, crear cliente, agregar cuentas, eliminar cuentas y eliminar cliente.
- Persistencia simple en `data/clients.json`.

Notas de seguridad:
- Este proyecto es para fines educativos y no debe usarse en producción sin añadir autenticación, validación de entradas y manejo de concurrencia en la persistencia.
